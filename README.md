# Groq Streamlit Chatbot

A simple LLM chatbot built with Groq, LangChain, and Streamlit.

## Setup

1. Clone the repo
2. Run `uv sync`
3. Create a `.env` file and add your Groq API key like this:
   GROQ_API_KEY=your_key_here
4. Run `uv run streamlit run app.py`

## Tech Stack

- Groq
- LangChain
- Streamlit
- uv
